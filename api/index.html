API Documentation

Get Questions

Endpoint to get questions with optional count parameter.

        GET /?count=5

Get All Categories

Endpoint to get a list of all categories.

        GET /quiz/cat/

Get Questions by Category

Endpoint to get questions by category with optional count parameter.

        GET /quiz/cat/:name?count=5

Add a Question

Endpoint to add a new question with options and category.

        POST /quiz/add

Request Body:

        {
            "question": "What is the capital of France?",
            "category": "Geography",
            "options": ["London", "Berlin", "Paris", "Madrid"],
            "answer": "Paris"
        }

Environment Variables

Ensure you have the following environment variables set:

        
DB_HOST=your_database_host
DB_USER=your_database_user
DB_NAME=your_database_name
DB_PASSWORD=your_database_password

Responses

The API returns JSON responses with appropriate status codes.

  • 200 OK - Successful response
  • 400 Bad Request - Invalid request
  • 500 Internal Server Error - Server error

Example Usage

        
const fetch = require('node-fetch');

// Fetch questions with a count of 10
fetch('http://localhost:8081/?count=10')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error(error));
